Capabilities

    
Every Layer,
Covered.

    


      

        01
        

          
        

        
Security Detection

        
Catches SQL injection, hardcoded secrets, and plain-text passwords — each severity-scored from 1 to 5.

      


      

        02
        

          
        

        
Logic Analysis

        
Off-by-one errors, wrong comparisons, missing null checks, and divide-by-zero traps across all supported languages.

      


      

        03
        

          
        

        
Multi-Language

        
Python, JavaScript, and Java — each with language-idiomatic bug patterns in the bug bank.

      


      

        04
        

          
        

        
Dual LLM Fallback

        
Claude Haiku runs first. Any exception silently falls back to GPT-4o-mini with no disruption to the API response.

      


      

        05
        

          
        

        
RL Reward Grading

        
Every review earns a reward score. +25 for critical security catches, −25 for missed criticals. The model learns.

      


      

        06
        

          
        

        
Review History

        
All reviews and training runs stored in SQLite. Query via /api/history and /api/stats endpoints.

      


    

  







  

    
REST API

    
Five Endpoints.
Zero Config.

    
Rate-limited to 20 req/min per IP. All responses are JSON. Start the server with python app.py.
